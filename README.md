# [JavaScript Design Patterns]
## Author: [Franco Queirolo][instructor url]

What's Inside This Repository:
This repository offers a clear explanation and practical examples of some of the most common and useful design patterns employed in JavaScript. To help you see them in action immediately, each pattern is demonstrated with a standalone, practical example, bundled within its own JavaScript module.

Dive in and take your software design skills to the next level!

## Source Files Description

1. Clone repository to your machine
    `git clone https://github.com/francoqueirolo/javascript-design-patterns`

2. Run Bower install
    `npm run dev`

3. Open index.html in your browser and open the browser's console

4. Use the global run() method to run one of the examples
    `run('factory')`

Valid example names (to be passed as a string to the run() function) are:

* `factory`
* `pubsub`
* `strategy`
* `observer`
* `cor`
* `mediator`
* `iterator`
* `facade`
* `decorator`
* `adapter`
* `composite`
* `proxy`
